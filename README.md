## Manual Testing Assignment

This repository contains the deliverables for the QA manual testing assignment.

## Repository Contents

### Manual_Test_Cases
- Detailed manual test cases
- Includes test scenario, test steps, expected result, and actual result
- Covers User Registration, Sign In, and Passkey flows

### Bug_Report
- List of identified defects
- Includes bug ID, title, module, description, steps to reproduce, severity, priority, and status

### Automation
- Basic Selenium automation script for Sign In functionality
- Includes setup instructions and steps to execute the tests
- Automation is limited to basic scenarios as required in the assignment

## Tools Used
- Manual Testing: Microsoft Excel
- Automation Testing: Selenium WebDriver with Java
- Build Tool: Maven

## Notes
- Test cases and bug reports are provided in Excel format
- Automation scripts are included along with execution instructions
